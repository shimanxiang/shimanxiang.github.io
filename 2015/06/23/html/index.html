XHTML与HTML的主要区别如下：

  1. XHTML要求正确嵌套
  2. XHTML所有元素必须关闭,包括空标签
  3. XHTML区分大小写，XHTML所有标记和属性都要小写，属性值必须加引号，属性不能简写。
  4. XHTML 文档必须拥有一个根元素
  5. XHTML用 id 属性代替 name 属性。
  6. 语言属性（lang）lang 属性应用于几乎所有的 XHTML 元素。它定义元素内部的内容的所用语言的类型。如果在某元素中使用 lang 属性，就必须添加额外的 xml:lang，像这样：

JavaScript 判定一个字符串是否包含另一个字符串方法：

  1. 正则 /^[\w][h][t][t][p]+[\w]$/；

  2. indexOf str.indexOf(searchValue[, fromIndex])

    indexOf() 方法返回指定值在字符串对象中首次出现的位置。从 fromIndex 位置开始查找，如果不存在，则返回 -1。

  3. lastIndexOf str.lastIndexOf(searchValue[, fromIndex])

    lastIndexOf() 方法返回指定值在调用该方法的字符串中最后出现的位置，如果没找到则返回 -1。从该字符串的后面向前查找，从 fromIndex 处开始。

  4. search str.search(regexp)

    search() 方法用于检索字符串中指定的子字符串，或检索与正则表达式相匹配的子字符串。

块级元素和行内元素

块级元素：div,p,form,ul,li,ol,dl,address,fieldset,hr,menu,table…..

行内元素：span,strong,em,br,img,input,label,select,textarea,cite,a….

两者区别：1：块级元素会独占一行，其高度自动填满其父元素宽度，行内元素一行并排显示
2：对于行内元素，需要注意：

- 设置宽度width无效
- 设置高度height无效，可以通过line-height来设置
- 设置margin只有左右margin有效，上下无效。
- 设置padding只有左右padding有效，上下无效。
对于块级元素，注意：
- 块级元素即使设置了宽度，仍然是独占一行的。
